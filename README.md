# 使用指南

从fofa下载对应数据重命名为**fofa.csv**

然后运行main.sh即可

link.txt   为节点连接

check.md  为节点信息

ip.txt    为成功登录的后台地址

bpb.txt bpb链接
```
搜索TLS站点 搜索关键词：icon_hash="-1354027319" && asn="13335" && port="443"
搜索noTLS站点 搜索关键词：icon_hash="-1354027319" && asn="13335" && port="80"

```
